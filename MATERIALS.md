# Materials System

This system governs the properties of objects, allowing objects to interact with any object with a certain property.</br>
For example, fire should only burn flammable objects.

## Properties
 - Flammable
 - Porous
 - Meltable
 - Brittle
 - Freezable
 - Dissolvable
 - Malleable
 - Conductive
 - Cuttable
